# CortexFlow (AI Browser Agent)

مشروع **CortexFlow** هو نظام متطور لوكلاء التصفح المعتمدين على الذكاء الاصطناعي، تم تحديثه وتحسينه ليعمل بكفاءة عالية على منصة **Render** مع دعم كامل للنماذج المحلية ومساحة تخزين 50 جيجابايت.

## المميزات الجديدة في CortexFlow:
- **دعم النماذج المحلية:** إمكانية تحميل وتشغيل نماذج مثل Llama-2 و Mistral و Qwen مباشرة.
- **إدارة قاعدة البيانات:** نظام SQLite مدمج لإدارة المهام وسجلات التنفيذ.
- **تحسين النشر:** ملف Dockerfile محسن يضمن تهيئة قاعدة البيانات وتحميل النماذج في الخلفية.
- **إعادة تسمية شاملة:** تم تحديث جميع المراجع لتتوافق مع الهوية الجديدة للمشروع.

## تعليمات النشر على Render:
1. قم بربط مستودع GitHub الخاص بك بمنصة Render.
2. اختر نوع الخدمة **Web Service**.
3. استخدم الإعدادات التالية:
   - **Runtime:** Docker
   - **Plan:** Starter (أو أعلى لدعم مساحة القرص)
   - **Disk:** أضف قرصاً (Disk) باسم `models-storage` بمسار `/app/models` وحجم **50GB**.
4. أضف المتغيرات البيئية التالية:
   - `RENDER_API_KEY`: (استخدم المفتاح الموفر من Render)
   - `GITHUB_TOKEN`: (استخدم التوكين الخاص بك من GitHub)
   - `USE_LOCAL_MODELS`: `true`
   - `DB_PATH`: `/app/data/app.db`
   - `MODELS_PATH`: `/app/models`

## كيفية تشغيل المشروع محلياً:
```bash
npm install
npm run init-db
npm start
```

## ملاحظات هامة:
- يتم تحميل نماذج الذكاء الاصطناعي في الخلفية عند بدء التشغيل لأول مرة لتجنب تأخير استجابة الخادم.
- تأكد من توفير مساحة القرص المطلوبة (50GB) في إعدادات Render لضمان نجاح تحميل النماذج.
