# منصة وكيل الذكاء الاصطناعي - النسخة المحسنة

## نظرة عامة

منصة متقدمة لأتمتة المتصفح والمهام باستخدام الذكاء الاصطناعي (DeepSeek). تجمع بين قوة Playwright للأتمتة وذكاء DeepSeek للتخطيط والتفكير الديناميكي.

## الميزات الجديدة

### 1. **Sidebar محسن للمهام**
- عرض قائمة المهام في شريط جانبي على اليمين
- إحصائيات سريعة (إجمالي، مكتملة، قيد التنفيذ، فشلت)
- تحديث فوري لحالة المهام
- اختيار المهام من الـ Sidebar للتركيز عليها

### 2. **صفحة السجلات المنفصلة**
- نافذة منفصلة تعرض السجلات الحية والتفكير معاً
- عرض تفكير الذكاء الاصطناعي في الوقت الفعلي (مثل Manus)
- تصميم ثنائي العمود: السجلات على اليسار، التفكير على اليمين
- تحديث تلقائي مع الألوان المختلفة لأنواع السجلات

### 3. **أتمتة DeepSeek الذكية**
- **اختيار النموذج التلقائي**: DeepSeek يحدد نوع المهمة (متصفح/نظام/تطوير) تلقائياً
- **تحديد الأولوية التلقائي**: تحديد أولوية المهمة بناءً على الوصف
- **تحليل المهام**: فهم عميق لمتطلبات المهمة قبل التنفيذ

### 4. **شاشة عرض محسنة (Manus-like)**
- عرض صغير قابل للتوسيع للمتصفح المباشر
- تفاعل سلس مع المتصفح (النقر والكتابة)
- حالة الاتصال الحية

### 5. **تفكير الذكاء الاصطناعي**
- عرض خطوات التفكير أثناء تحليل المهام
- تفكير خاص بكل خطة تنفيذ
- مساعدة في فهم قرارات الوكيل

### 6. **قدرات محسنة للمهام المعقدة**
- **تحديد الاستراتيجية التلقائية**: اختيار أفضل نهج تنفيذ
- **معالجة الأخطاء الذكية**: إعادة محاولة مع تكيف ديناميكي
- **تحليل الأنماط**: تعلم من الأخطاء والنجاحات السابقة
- **مهلة زمنية ديناميكية**: تعديل المهلة الزمنية بناءً على الحاجة

## المتطلبات

### متغيرات البيئة

```bash
# مفتاح DeepSeek API (اختياري - يفعل الميزات الذكية)
DEEPSEEK_API_KEY=your_api_key_here

# منفذ الخادم (افتراضي: 3000)
PORT=3000

# بيئة التطوير (development/production)
NODE_ENV=development

# مستوى السجلات (info/debug/error)
LOG_LEVEL=info
```

### التثبيت

```bash
# استنساخ المشروع
git clone https://github.com/melsofany/CortexFlow.git
cd CortexFlow

# تثبيت الاعتماديات
npm install

# تثبيت متطلبات Playwright
npx playwright install-deps

# إنشاء ملف .env
cp .env.example .env
# ثم أضف مفتاح DeepSeek API الخاص بك
```

## الاستخدام

### بدء الخادم

```bash
# وضع التطوير
npm run dev

# وضع الإنتاج
npm start
```

الخادم سيبدأ على `http://localhost:3000`

### واجهة المستخدم

#### الـ Sidebar
- عرض قائمة المهام الحالية
- إحصائيات سريعة للمهام
- انقر على مهمة لتحديدها

#### نموذج إرسال المهام
1. اكتب وصف المهمة
2. اختر نوع المهمة (سيتم التحديد التلقائي إذا تركت "تلقائي")
3. اختر الأولوية (سيتم التحديد التلقائي إذا تركت "تلقائي")
4. انقر "إرسال المهمة"

#### عرض السجلات
- انقر على زر 📋 في الزاوية العلوية اليسرى
- شاهد السجلات الحية والتفكير معاً
- أغلق النافذة بالنقر على X أو خارج النافذة

#### التحكم بالمتصفح
- انقر على شاشة المتصفح للتفاعل
- استخدم لوحة المفاتيح للكتابة
- شاهد الحالة الحية في الزاوية السفلية اليمنى

## استكشاف الأخطاء

### الخادم لا يبدأ
- تحقق من تثبيت Node.js
- تأكد من عدم استخدام المنفذ 3000
- تحقق من السجلات للأخطاء التفصيلية

### المتصفح لا يعمل
- تأكد من تثبيت Playwright: `npx playwright install`
- تحقق من توفر الذاكرة
- جرب مع `headless: false` للتصحيح

### DeepSeek لا يعمل
- تحقق من مفتاح API
- تأكد من الاتصال بالإنترنت
- تحقق من حد الطلبات
